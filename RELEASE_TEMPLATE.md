## ترقية وتشغيل (Deployment / Upgrade)
### متطلبات
- PHP: `>= 8.2`
- Laravel: `12.x`

### خطوات الترقية المقترحة
1) أخذ نسخة احتياطية من قاعدة البيانات والملفات (خصوصاً `storage/`).
2) تحديث الكود على السيرفر.
3) تثبيت الاعتماديات:

```bash
composer install --no-dev --optimize-autoloader
```

4) تنفيذ الترحيلات:

```bash
php artisan migrate --force
```

5) تجهيز التخزين العام (إن لم يكن موجوداً):

```bash
php artisan storage:link
```

6) تنظيف/بناء كاش Laravel:

```bash
php artisan optimize:clear
php artisan config:cache
php artisan route:cache
php artisan view:cache
```
